# Daily Habit Tracker

## Features
- Tracks Fajr, Quran, workout, Python study, daily thought
- Saves to file with date
- View full history
- Input validation + error handling

## How to run
python habit_tracker.py

## About
Built this for my own daily tracking — Fajr prayer, 
Quran reading, fitness, and learning. 
Wanted something simple I'd actually use every day.

## What I learned
- File handling, dictionaries, functions, try/except, while loops
